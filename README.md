# README FOR THE H2GO Project 

## dependencies used 
- springdoc for API documentation with swagger-ui
  - resource: https://www.baeldung.com/spring-rest-openapi-documentation
  - you can access it via "http://localhost:8080/swagger-ui/index.html#/" after starting the application"
- dotenv-java for loading environment variables from a .env file automatically
  - resource: https://github.com/paulschwarz/spring-dotenv
- lombok for reducing boilerplate code in Java classes
- spring actuators for monitoring and managing the application
  - resource: https://www.baeldung.com/spring-boot-actuators
- api versioning 
  - resource: https://www.baeldung.com/spring-api-versioning
